## 基本介绍
HPC(High Performance Computing)是高性能计算的缩写，是一种利用计算机集群进行高速计算的技术。HPC的应用领域非常广泛，包括科学计算、工程计算、数据挖掘、人工智能等。HPC的核心是并行计算，通过将计算任务分解成多个子任务，然后在多个计算节点上并行执行，从而提高计算速度。

## HPCer基础技能
* 编程语言
  * C/C++
  * Fortran
  * Python
* 并行模型
    * MPI
    * OpenMP
    * CUDA
* 计算机系统基础
    * 进程的创建和通信
* linux基础、shell编程
* 线性代数
* 模型优化方法
* 各种高效工具
